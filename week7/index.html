Something that I didn't know before but now I know that I can apply are the recursive functions that can invoke themselves inside until a certain condition is met, the most common example is with factoring since it is a more interactive process:


function factorial(n) {
if (n === 0) {
return 1;
} else {
return n * factorial(n - 1);
}
}
