Something that I know I can use in the future is to use set and get items from the localStorage as shown in the example below.
Storage.getItem(): returns the value of a key.
Storage.setItem(): is for when we want to designate a key and a value or reuse a key and update the previous value.

function loadStory(){
var storyName = document.getElementById(“name_input”).value
var storyHTML = localStorage.getItem(storyName)
document.getElementById(“story_editor”).value = storyHTML
}
function saveStory(){
var storyName = document.getElementById(“name_input”).value
var storyHTML = document.getElementById(“story_editor”).value
localStorage.setItem(storyName, storyHTML)
}
